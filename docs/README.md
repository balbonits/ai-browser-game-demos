# Docs

Project documentation for the AI Browser Game Demos repo.

## Index

- [`adding-a-game.md`](adding-a-game.md) — step-by-step guide for creating a new game in this repo.
- [`pixellab.md`](pixellab.md) — how to use the PixelLab MCP to generate characters, animations, and tilesets.
- [`conventions.md`](conventions.md) — code style, file layout, and naming conventions.
- [`agents.md`](agents.md) — the `artist` and `dev` subagent personas, when to use each.
- [`audio.md`](audio.md) — Web Audio conventions (synthesized, no audio files).
- [`deploy.md`](deploy.md) — how to deploy to Vercel as a subsite of `jdilig.me`.
- [`games/`](games/) — per-game documentation (one file per game, plus a template).

## Purpose

This folder holds workflows, conventions, and notes that are too long for the root `README.md` / `CLAUDE.md` / `AGENTS.md` files. Treat root markdown files as the entry points; this folder is the reference material behind them.
